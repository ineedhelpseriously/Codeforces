# 🚀 Codeforces Problem Solving Roadmap (Python)

This roadmap is designed for improving **competitive programming on Codeforces using Python**.  
Topics are ordered by **difficulty progression** and **real CF usage**, not theory.

---

## 🟢 Level 1: Absolute Foundation (800)

**Goal:** Solve A problems confidently and quickly.

### Topics to Learn

- Fast Input / Output (`sys.stdin.readline`)
- Loops and Conditionals
- Basic Math (`%`, `//`, `abs`, `min`, `max`)
- Strings (indexing, slicing, `.count()`)
- Sorting (`sorted()`, `key=`)

### Common CF Usage

- Simple implementation
- Straightforward greedy logic
- Single-pass problems

✅ **Move on when:**  
A problems feel trivial and take <10 minutes.

---

## 🟡 Level 2: Core Problem Solving (900–1000)

**Goal:** Start clearing B problems consistently.

### Topics to Learn

- Prefix Sums
- Frequency Counting (`dict`, `Counter`)
- Greedy Basics
- Two Pointers
- Simple Simulation

### Common CF Usage

- Array & string manipulation
- Counting and comparisons
- Constraint-based logic

✅ **Move on when:**  
Most B problems are solvable with ≤1 wrong attempt.

---

## 🟠 Level 3: Optimization Thinking (1100–1200)

**Goal:** Avoid TLE and write efficient Python solutions.

### Topics to Learn

- Time Complexity Intuition
- Binary Search (`bisect`)
- Sets & Maps
- Greedy Proof Reasoning
- Math Tricks (parity, GCD, divisors)

### Common CF Usage

- Constraint pruning
- Search space reduction
- Editorial-driven logic

✅ **Move on when:**  
You can solve some C problems (even if slowly).

---

## 🔵 Level 4: Advanced Python CF (1300–1400)

**Goal:** Be contest-competitive using Python.

### Topics to Learn

- Sliding Window
- Heaps (`heapq`)
- Recursion (DFS basics)
- Bit Manipulation
- Constructive Algorithms

### Common CF Usage

- Scheduling problems
- Interval logic
- Hard B / C problems

---

## 🔴 Level 5: Optional but Powerful (1400+)

**Goal:** Push Python to its limits on CF.

### Topics to Learn

- Dynamic Programming
- Graph Algorithms (BFS / DFS)
- Number Theory
- Coordinate Compression
- Python Micro-optimizations

---

## 🧠 Practice Strategy

- **Weekdays:** 1–2 B problems
- **Weekends:** Attempt 1 C problem (read editorial if stuck)
- Always analyze **why** your solution works or fails.

> ⚠️ If a solution feels slow, redesign the logic before optimizing syntax.

---

## 🎯 Milestones

- 100+ solved → Strong foundation
- 150+ solved → Contest ready
- Regular Div.3 participation recommended

---

Happy Coding 🚀
